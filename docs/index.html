SOS - Suporte à Operação de Sistemas Elétricos

Aplicação destinada a gestão, controle de versão, indexação e pesquisa de documentos relacionados à operação do sistema elétrico brasileiro.
Estão inclusos atualmente na relação de documentos disponíveis na aplicação:

  • Manual de Procedimentos da Operação (MPO - Procedimentos de Rede Módulo 10 e submódulos)
  • Mensagens de Operação (MOPs)
  • Diagramas Unifilares Rede de Operação
  • Diagramas Unifilares Rede de Supervisão
  • Diagramas Operativos Hidráulicos

Como Instalar

Última versão disponível do software para uso está disponível aqui.

Para utilizar realize o download do arquivo .zip, extraia o conteúdo para uma pasta desejada e execute o arquivo SOS.exe
O software tem por objetivo ser portátil, portanto não necessitar instalação facilitando seu uso em sala de controle.

Pré-requisitos

  • .NET Framework 4.6
  • Microsoft Visual C++ 2013

Como Funciona

Ao iniciar a aplicação irá solicitar confirmação de credenciais da plataforma CDRE ONS. É possível utilizar a aplicação sem conexão com à Internet/Intranet pelo botão Entrar Offline, entretanto nesta opção não será possível download ou atualização de quaisquer documentos.

Ao efetuar login a interface de visualização de documentos irá ser carregada e a rotina de atualização/indexação de documentos irá iniciar.

Ao concluir atualização o histórico estará disponível no link

AVISO LEGAL

O Operador Nacional do Sistema Elétrico (ONS) não tem qualquer participação com este projeto e a autenticação com sua plataforma CDRE ONS é de caráter autorizativo individual para possibilitar o download de documentos que o nível de acesso do usuário garante. A autenticação é necessária para download de quaisquer documentos, independente se disponível ao público, como o MPO, uma vez que a aplicação é destinada à usuários do setor elétrico que não raramente são credenciados à plataforma.
